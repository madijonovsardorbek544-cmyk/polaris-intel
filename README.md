# POLARIS — Political & Cyber Risk Intelligence System

POLARIS is a unified cyber–geopolitical intelligence platform that analyzes cyber threats,
disinformation signals, and diplomatic risk indicators in one system.

## Core Modules
- Cyber Threat Intelligence
- Disinformation & Election Monitoring
- Cyber Diplomacy & Sanctions Analysis
